# Local Inference

**Run the model. Own the stack.**

A cinematic marketing concept site exploring what it actually looks like to run serious local LLMs on personal and prosumer hardware — without renting intelligence from someone else's data center.

## Live Demo

**[https://loktar00.github.io/grok-heavy-test/](https://loktar00.github.io/grok-heavy-test/)**

## What This Is

This is a high-end, tongue-in-cheek marketing concept built as a single self-contained HTML experience. It walks through five real hardware paths people actually use for local inference in 2026:

- NVIDIA RTX 3060 (the approachable entry point)
- NVIDIA RTX 3090 (the legendary used-market workhorse)
- NVIDIA RTX PRO 6000 Blackwell (the serious professional workstation)
- NVIDIA DGX Spark (the compact personal AI supercomputer)
- MacBook Pro with unified memory (the elegant, portable option)

The hero features a sequential cinematic video experience that plays each device's story one after another.

## Features

- Beautiful sequential video playlist (no giant single video file)
- Clean, modern header with the actual headline
- Thoughtful hardware comparison with "Insanity Index"
- Strong, non-fluffy copy that respects the local AI community
- NVIDIA-inspired green and black aesthetic
- Fully responsive

## Running Locally

Just open `index.html` in any modern browser. No build step, no dependencies.

```bash
open index.html
```

## Tech Stack

- Single-file HTML
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Custom-generated video assets and imagery

## Credits
